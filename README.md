## IBM Data Science Professional Certificate Coursework

### 1. What Is Data Science?

- **Provider:** IBM through Coursera
- **Completed:** May 18, 2020
- **Grade:** 92.80%
- **Estimated coursework:** 11 hours
- **Certificate:** [View official certificate (PDF)](certificates/01-what-is-data-science.pdf)

#### What I Learned

- Defined data science and its importance in a data-driven world.
- Explored career paths and professional roles within data science.

- Examined the growing importance of data science across modern industries.

Data literacy, data-driven decision-making, data analysis concepts, data mining, big data, machine learning concepts, deep learning concepts, cloud computing, and data storage fundamentals.

### 2. Tools for Data Science

- **Provider:** IBM through Coursera
- **Completed:** June 21, 2020
- **Grade:** 93.25%
- **Estimated coursework:** 16 hours
- **Certificate:** [View official certificate (PDF)](certificates/02-tools-for-data-science.pdf)

#### What I Learned

- Identified the principal tools, libraries, packages, datasets, and machine-learning models used by data scientists.
- Explored programming languages commonly used in data science, including Python, R, and SQL.
- Developed working knowledge of Jupyter Notebooks and RStudio.
- Learned how Git and GitHub support source-code management, version control, and collaboration.

#### Skills Developed

Jupyter Notebooks, Python, R, SQL, Git, GitHub, version control, cloud platforms, data-visualization tools, Scikit-learn, and software-development tools.

#### Cybersecurity Relevance

This course introduced technical tools that transfer directly to cybersecurity analyst work. Python and Jupyter Notebooks can support security-data analysis and automation, SQL can be used to query security records, and GitHub provides version control for scripts, detection rules, documentation, and portfolio projects.
